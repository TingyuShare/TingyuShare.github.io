# Markdown 用法指南

Markdown 是一种轻量级的标记语言，广泛用于格式化文本。它的语法简单易学，适合编写文档、博客、README 文件等。本文将介绍 Markdown 的基本语法和常用功能，帮助你快速上手 Markdown。

## 标题

Markdown 使用 `#` 符号表示标题。`#` 的数量代表标题的级别，最多支持六级标题。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 段落与换行

段落之间需要空一行。如果需要在同一段落内换行，可以在行尾添加两个空格。

```markdown
这是第一段。

这是第二段。  
这是同一段内的换行。
```

**效果：**

这是第一段。

这是第二段。  
这是同一段内的换行。

## 文本样式

Markdown 支持以下文本样式：

- **粗体**：用 `**` 或 `__` 包裹文本。
- *斜体*：用 `*` 或 `_` 包裹文本。
- ~~删除线~~：用 `~~` 包裹文本。

```markdown
**粗体**  
*斜体*  
~~删除线~~
```

**效果：**

**粗体**  
*斜体*  
~~删除线~~

## 列表

### 无序列表

使用 `-`、`*` 或 `+` 创建无序列表。

```markdown
- 项目一
- 项目二
  - 子项目
```

**效果：**

- 项目一
- 项目二
  - 子项目

### 有序列表

使用数字加点创建有序列表。

```markdown
1. 第一项
2. 第二项
   1. 子项
```

**效果：**

1. 第一项
2. 第二项
   1. 子项

## 链接与图片

### 链接

使用 `[文本](URL)` 创建链接。

```markdown
[Google](https://www.google.com)
```

**效果：**

[Google](https://www.google.com)

### 图片

使用 `![替代文本](图片URL)` 插入图片。

```markdown
![Markdown Logo](https://markdown-here.com/img/icon256.png)
```

**效果：**

![Markdown Logo](https://markdown-here.com/img/icon256.png)

用 `` ` `` 包裹行内代码。

## 表格

使用 `|` 和 `-` 创建表格。

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
```

**效果：**

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |

## 高级功能

### 任务列表

使用 `- [ ]` 和 `- [x]` 创建任务列表。

```markdown
- [x] 完成任务一
- [ ] 完成任务二
```

**效果：**

- [x] 完成任务一
- [ ] 完成任务二

### 内嵌 HTML

Markdown 支持内嵌 HTML，用于实现更复杂的格式。

```markdown
<p style="color:red;">这是红色的文字。</p>
```

**效果：**

<p style="color:red;">这是红色的文字。</p>

## 总结

Markdown 是一种简单而强大的标记语言，适用于各种文档编写场景。通过掌握以上语法，你可以轻松创建格式化的文本内容。如果需要更高级的功能，可以结合 HTML 或其他工具扩展 Markdown 的能力。
